# Deploy Gratuito

## Frontend

O frontend é estático e pode ser publicado no GitHub Pages ou Netlify.

## Backend

O backend Express pode ser publicado gratuitamente em serviços como:

- [Render](https://render.com)
- [Railway](https://railway.app)
- [Fly.io](https://fly.io)

## Documentação

A documentação MkDocs pode ser publicada como site estático via GitHub Pages.

## Passos gerais

1. Criar repositório no GitHub.
2. Enviar o código para o repositório.
3. Configurar GitHub Pages ou outro serviço para o frontend.
4. Configurar serviço gratuito para a API.
5. Publicar MkDocs como site estático.

## Exemplo de link de documentação

- `https://<usuário>.github.io/<repositório>/`
