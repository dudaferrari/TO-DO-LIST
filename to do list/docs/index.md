# Todo List MVC

Bem-vindo ao projeto Todo List com arquitetura MVC e mono-repo.

Este projeto implementa uma lista de tarefas que permite cadastrar, remover, marcar como concluída e adicionar lembretes.

## Estrutura do projeto

- `packages/api`: backend em Express com armazenamento em memória.
- `packages/web`: frontend estático com HTML, CSS e JavaScript.
- `docs`: documentação MkDocs.
- `speckit.constitution`: documento de justificativa e escolhas arquiteturais.
